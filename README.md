# Күй аңызы

Интерактивті жоба по рассказу Тахауи Ахтанова.

Что умеет:
- показывает художественную веб-сцену;
- использует Python + MediaPipe для распознавания пальцев через камеру;
- `1 палец` включает первый трек;
- `2 пальца` включает основной күй;
- через `3` секунды после правильного кюя верблюд начинает давать молоко.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python app.py
```

После запуска откройте:

```text
http://127.0.0.1:4173/
```

На странице один раз нажмите кнопку включения звука, затем показывайте в камеру `1` или `2` пальца.
