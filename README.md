# Күй аңызы

Интерактивті жоба по рассказу Тахауи Ахтанова.

Что умеет:
- показывает художественную веб-сцену;
- использует MediaPipe Hand Landmarker в браузере через JavaScript;
- во всех камерах показывает только руку/пальцы на тёмном фоне, без лица и окружения;
- `1 палец` включает первый трек;
- `2 пальца` включает основной күй;
- через `3` секунды после правильного кюя верблюд начинает давать молоко.
- на странице `/test` запускает тест по «Күй аңызы», где ответы можно выбирать кнопками или жестами `1-4` пальца.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python app.py
```

После запуска откройте:

```text
http://127.0.0.1:4173/
```

Дополнительно доступны страницы:

```text
http://127.0.0.1:4173/dombra
http://127.0.0.1:4173/test
```

После запуска разрешите браузеру доступ к камере, один раз нажмите кнопку включения звука, затем показывайте в камеру `1` или `2` пальца.

## Что изменилось

- сервер больше не обрабатывает видео и не запускает Python MediaPipe;
- распознавание руки теперь идёт на фронтенде через `@mediapipe/tasks-vision`;
- Flask нужен для отдачи HTML, JS и аудио-треков.

Примечание: фронтенд загружает MediaPipe Web runtime и модель жестов по сети, поэтому для загрузки страницы нужен доступ в интернет. Если нужен полностью офлайн-режим, bundle/wasm/model стоит положить в проект локально.
